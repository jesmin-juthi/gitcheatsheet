# gitcheatsheet

### Basic Commands For New Repo

- echo "# gitcheatsheet" >> README.md
- git init
- git add README.md
- git commit -m "first commit"
- git branch -M main
- git remote add origin https://github.com/user/repo.git
- git push -u origin main


### Git Commands For Existing Repo

- git remote add origin https://github.com/user/repo.git
- git branch -M main
- git push -u origin main
